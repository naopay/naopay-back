- WebAuthn : only Windows Hello for now (fingerprints for the future)

- NestJS
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
